# Blinkit Sales Dashboard

## Project Overview
This Power BI dashboard analyzes Blinkit sales performance and customer ordering trends.

## Key Metrics
- Total Orders: 122
- Total Quantity: 612
- Total Revenue: 59.81K
- Average Delivery Time: 34.95 Minutes
- Average Rating: 3.01

## Dashboard Features
- Revenue Analysis
- City-wise Orders
- Category Analysis
- Payment Method Analysis
- Monthly Sales Trends
- Product Performance

## Tools Used
- Power BI
- Excel
- DAX
- Power Query
